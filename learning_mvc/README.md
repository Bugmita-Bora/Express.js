# Airbnb Clone - Express.js

A full-stack web app built with Node.js and Express.js

## Features
- Home page with navigation
- Host registration form
- Form data parsing with Express middleware
- Custom 404 error page
- MVC-style folder structure (routes, views, utility)

## Tech Stack
Node.js, Express.js, HTML, CSS

## How to run
npm install
node app.js
